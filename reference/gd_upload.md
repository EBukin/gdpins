# Upload a local file to Drive

Upload a local file to Drive

## Usage

``` r
gd_upload(adapter, local_path, path)
```

## Arguments

- adapter:

  A `gdpins_drive_adapter` object.

- local_path:

  Character scalar. Path to the local source file.

- path:

  Character scalar. Destination path relative to the adapter root.

## Value

`invisible(adapter)`.
