# Download a Drive file to a local path

Download a Drive file to a local path

## Usage

``` r
gd_download(adapter, path, local_path)
```

## Arguments

- adapter:

  A `gdpins_drive_adapter` object.

- path:

  Character scalar. Source path relative to the adapter root.

- local_path:

  Character scalar. Destination local path.

## Value

`invisible(local_path)`.
