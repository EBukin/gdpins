# Board initialisation and S3 methods for gdpins_board

Provides
[`gdpins_init_board()`](https://ebukin.github.io/gdpins/reference/gdpins_init_board.md)
(the universal board constructor) and S3 methods for
[`print()`](https://rdrr.io/r/base/print.html),
[`summary()`](https://rdrr.io/r/base/summary.html), and
[`format()`](https://rdrr.io/r/base/format.html) on `gdpins_board`
objects.
